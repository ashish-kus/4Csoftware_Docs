tolower()

### tolower()

Purpose:

tolower() converts any upper case letters in an alpha variable to lower
case.

Usage:

aret = tolower(\<avar\>);

Arguments:

alpha \<avar\> - The alpha variable to convert.

Returns:

alpha \<aret\> - The converted alpha.

Where Used:

tolower() can be called from anywhere.

Example:

Description:

tolower() translates an alpha variable by converting all upper case
letters to lowercase. All other characters are left untouched. The
converted alpha is the return value of tolower(). The original alpha is
left unchanged unless it is specified as \<aret\>.
