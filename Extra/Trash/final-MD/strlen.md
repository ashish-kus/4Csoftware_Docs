## strlen()
### Purpose
strlen() returns the length of an alpha variable.

### Usage
len = strlen(<avar>);

### Arguments
alpha <avar> - The field to get the length of.

### Returns
integer len - the length of <avar>

### Where Used
strlen() can be called from anywhere.

### Example
None

### Description
strlen() returns the length of an alpha field.
