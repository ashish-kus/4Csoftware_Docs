sys.flush()

### sys.flush()

Purpose:

sys.flush() flushes ALL files and calls sync().

Usage:

sys.flush();

Arguments:

NONE

Returns:

NONE

Where Used:

sys.flush() can be called from anywhere.

Example:

Description:

sys.flush() flushes ALL files and calls sync().

Bugs/Features/Comments:

Currently unimplemented with Sequential Files
