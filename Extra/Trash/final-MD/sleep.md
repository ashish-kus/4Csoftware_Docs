## sleep()
### Purpose
sleep() pauses for \<count\> seconds.

### Usage
```c
sleep(<count>);
```

### Arguments
integer \<count\> - The number of seconds to sleep for.

### Returns
sleep always returns 0.

### Where Used
sleep() can be called from anywhere.

### Example
```c
sleep(3);
```

### Description
sleep() pauses for \<count\> seconds.
