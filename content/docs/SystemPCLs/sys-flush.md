## sys.flush()
### Purpose
sys.flush() flushes ALL files and calls sync().

### Usage
```c
sys.flush();
```

### Arguments
NONE

### Returns
NONE

### Where Used
sys.flush() can be called from anywhere.

### Example
None

### Description
