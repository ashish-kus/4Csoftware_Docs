## sys.get_pid()
### Purpose
sys.get_pid() returns the Process Id of the 4C process.

### Usage
```c
pid = sys.get_pid();
```

### Arguments
None

### Returns
integer <pid> - The Process Id

### Where Used
sys.get_pid() can be called from anywhere.

### Example
None

### Description
